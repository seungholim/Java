set 과 HashSet

1. set
	인터페이스
	자바의 컬렉션 프레임워크에서 중복 요소를 허용하지 않ㅇ는 데이터 구조
	순서 보장하지 않음
	각 요소는 고유해야 함 (중복된 요소를 허용하지 않기 때문~)
	
2. HashSet
	set 인터페이스를 구현한 클래스
	중복 요소를 허용하지 않고, 순서를 보장하지 않는 집합 데이터 구조 제공
	중복을 제거하고 고유한 요소를 저장할 때 사용
	내부적으로 해시 테이블을 사용하여 데이터를 저장함
	검색 추가 제거 작업이 빠름
	
**Hash란?
	입력 데이터를 고정 길이의 문자열로 변환하는 알고리즘
	변환된 문자열 자체를 가리키는 용어
	해시 함수는 임의의 크기의 입력 데이터를 고정 크기의 출력 데이터로 변환
	변환된 출력 데이터 = 해시 값 혹은 해시 코드라고 부름
