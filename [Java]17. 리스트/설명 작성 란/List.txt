List
	데이터를 순서대로 저장하는 자료 구조 중 하나
	여러 개의 요소를 포함할 수 있음
	배열(Array)과 달리 크기를 (유)동적으로 조절할 수 있으며, 중복된 요소를 허용할 수 있음
	
	Java의 util 패키지에 속해있음 (누군가 만들어놓은 객체 및 오브젝트)
	List 인터페이스의 구현체는 ArrayList와 LinkedList
	Lost 인터페이스의 객체는 두 종류 
					1)ArrayList
					2)LinkedList
	
List와 Array의 차이점
공통점: 둘 다 데이터를 저장하는 자료구조.
	
	Array: 예약해서 가야하는 가게 (20명 예약했으면 딱 그만큼만 오셔)
		배열을 생성할 대 크기를 지정 > 이후 크기 변경 불가
		배열의 크기를 변경하려면 새로운 배열을 생성하고 데이터를 복사해야함
		특정 데이터 유형의 요소만 저장 가능 ex) 정수 배열 > 정수만 저장 가능	
	
	List: 일단 파티 주최해놓고 올 사람 오고 갈 사람 가셔
 		크기가 가변적(바뀌거나 바꿀 수 있음)
		필요에 따라 요소를 추가하거나 제거할 수 있음
		크기 조정이 자동으로 처리되어 여러 다른 유형의 요소를 저장할 수 있음
		> 혼합 데이터 유형을 저장하는데 사용할 수 있음
