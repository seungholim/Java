Graph(그래프)
그래프가ㅣㅇ렁ㄹ그래퍼ㅡㄹ아그래퍼ㅡㅍ프거르패그리않ㅈ댜ㅐㄹ긒래파ㅓㄱ르패그래프

	그래프를 만들고 다루는 것은 데이터 구조와 알고리즘에 관심이 있는 프로그래머에게 중요한 기술..
	여러 노드(또는 정점)와 그 노드들을 연결하는 엣지(Edge) a.k.a 간선 으로 구성됨
	네트워크, 지도, 소셜 네트워크, 경로 찾기 등에 사용
	
	인접리스트(Adjacency List) :
		각 노드마다 해당 노드와 연결된 다른 노드들을 리스트로 저장하는 방식
