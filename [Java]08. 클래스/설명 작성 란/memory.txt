
Memory(메모리)란?
	> 컴퓨터가 정보를 저장하고 처리하는 공간

**메모리의 종류 세 가지
	
	1. Static Memory(스태틱 메모리): 컴퓨터가 프로그램의 기본 정보와 데이터를 저장하는 곳
		이 정보는 프로그램이 실행되는 동안 계속 유지됨
		프로그램 안에 공유하는 중요한 정보는 여기에 저장
		ex) static void main() {} > stack에 저장(?)
		main: 파일 안에서 실행이 되는 곳 (main이 아니면 실행은 안되고 작성은.. 가능)
		
		
	2. Heap Memory (힙 메모리)
		컴퓨터가 프로그램이 필요할 때마다 새로운 데이터를 저장하는 곳
		이 데리터는 객체라고 부르며, 프로그램 실행 중에 계속해서 생기고 사라짐
		ex) 사용자의 정보나 게임 캐릭터 정보를 정장하는데 사용
	
	3. Stack Memory
		프로그램이 함수나 매서드를 실행할 때 사용하는 곳
		각 함수에는 자체적인 공간이 있어서, 함수가 실행되는 동안 필요한 정보를 저장
		함수가 끝나면 해당 정보는 지워짐..
--------------------------------------------------------------------------

**메모리 구분 이유

	1. 프로그램이 정볼르 보관하고 처리하는 방법을 이해하는데 도움
	2. 메모리를 효율적으로 사용하면 프로그램이 빠르고 안정적으로 작동
	3. 메모리를 잘 관리하면 프로그램 crash 되는 것을 방지
		(crash: 컴퓨터 프로그램 시스템이 예상하지 못한 오류 또는 결함으로 인해 비 정상적으로 종료되거나 작동하지 않은 상태)
