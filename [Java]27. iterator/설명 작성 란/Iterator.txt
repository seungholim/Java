Iterator
	컬렉션 요소를 순회하고 각 요소에 접근하는데 사용되는 인터페이스
	List, Set, Map 등과 같은 컬렉션 인터페이스를 순차적으로 처리할 때 유용
	
	iterator() : 초기화 메서드
	
	hasNext() : iterator가 순회할 다음 요소가 있는지 검사하는 메서드. boolean 타입을 반환한다
		> 다음 요소가 있으면 'true' / 더 이상 순회할 요소가 없으면 false 반환
		
	next() : 다음 요소로 이동하고 해당 요소를 반환하는 메서드
			 메서드를 호출하기 전에 hasNext를 사용하여 요소가 있는지 확인 필요
			 if 요소가 없는 상태에서 next를 사용하면 ..? > 예외 발생
			 머 그러면 exception 사용하셔도 되긴합니다 쩝스
	
	remove() : '현재 요소' 삭제
