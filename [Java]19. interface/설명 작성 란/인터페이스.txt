	클래스와 클래스간의 계약을 정의하는 추상 데이터 유형
	클래스의 행동을 정의하고,
	클래스가 특정 메서드를 반드시 구현해야 함을 강제
	메서드 선언만 포함
		메서드의 이름, 매개변수 및 반환 유형만 선언
		메서드의 구현 내용은 제공하지 않음
	
	추상 메서드'만' 포함한다
		= 모든 메서드는 추상 메서드로 선언되어있기 때문에 메서드의 본문이 없고,
		  메서드를 호출하면 해당 인터페이스를 구현하는 클래스에서 정의된 메서드 실행

		여러 인터페이스에서 정의된 메서드를 동시에 사용할 수 있음
