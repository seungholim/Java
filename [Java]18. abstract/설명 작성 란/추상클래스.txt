추상클래스(Abstract)
	일반적인 클래스와 다르게 직접 객체를 생성할 수 없음
	주로 다른 클래스들이 공통으로 가져야하는 메서드나 필드를 정의하는데 사용
	코드의 재사용성과 유지보수성을 향상시킬 수 있음
	
추상클래스 선언 방법:
	abstract 클래스 선언 앞에 붙임
	ex)
	abstract class MyAbstractClass {
		//추상메서드와 일반 필드, 메서드 등을 포함할 수 있음
		abstract void myAbstractMethod(); //추상메서드
	}

추상메서드
	추상클래스는 하나 이상의 추상메서드를 포함해야함
	이때 추상메서드는 선언만 있고 메서드의 구현은 없음
	> 반드시 하위 클래스에서 구현
